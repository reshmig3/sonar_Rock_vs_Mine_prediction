# mine_prediction using Machine Learning

This project predicts whether an underwater object is a mine or a rock using sonar signal data.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab

## Project Workflow
1. Load sonar dataset
2. Data preprocessing
3. Train-test split
4. Model training
5. Prediction and testing

## Result
The model predicts whether the object is:
- Mine
- Rock
